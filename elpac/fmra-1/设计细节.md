# 设计细节


## 主要技术

+ 力反馈电机控制:

  FOC BLDC 直流无刷电机矢量控制

  逻辑器件基于 FPGA

  通信使用 塑料光纤 (主控 FPGA <==> 局部电机控制 FPGA)

+ 电机旋转编码器: 复合编码器

  - 高精度低速 光学正交相对位置编码器

    塑料光纤信号回传 (A, B, Z 等 3 根光纤)

    局部 LED 照明

    位于刻度盘 外圈

  - 低精度高速 光学正交相对位置编码器

    类似同上, 位于刻度盘 内圈

  - 电容 低精度 绝对位置编码器

    2 根电线 (测试高频信号线)

+ 遥铠空间转换

  驾驶员空间泡: 相当于驾驶员变胖了, 用于遥铠终端接触自身的动作, 此时驾驶员尚未接触自身.

  默认空间泡保护层厚度: 2cm (突破 1cm 即 过载警告)

+ 驾驶员身体外形自动扫描 (3D 建模)

+ 防水: TODO

+ 能源: 锂电池, 钠离子电池, 有线供电, 无线供电

## 遥铠终端 组件位置

(一般情况) 位置从上至下的顺序:

+ 无线网络及媒体系统 (视觉/听觉): 头部 (头顶天线)

+ 工作状态指示灯: 额头

+ 调试接口 (USB-C): 头后方

+ 急停按钮: 后脑勺

+ 黑匣子 (手机): 胸前

+ 身体运动控制器: 胸腔

+ 手部运动控制器: 小臂

+ 无线充电天线位置: 背部 (靠墙站立)

+ 充电插口位置: 小腹 (肚脐)

+ 电源 (BMS): 骨盆

+ 电池组: 腿部 (大腿)


## 遥铠终端 工作状态指示灯

默认配置:

+ 黑色: 关机或休眠

+ 蓝色呼吸灯: 待机 (等待连接)

+ 蓝色常亮: 正常工作状态

+ 红色闪烁: (以及嘀嗒提示音) 电池电量低

+ 红色常亮: 工作状态异常/故障

+ 黄色: 正在充电
