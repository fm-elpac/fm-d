# FMCU-1
胖蚊子微控制器 (MCU)

+ FMTM 胖蚊子高精制造 100nm CMOS 工艺

+ RISC-V CPU: `rv32imc` (M, U 模式)

+ CHISEL 硬件描述语言

+ 主要性能参数:

  - 最高主频: 100MHz

  - 256kB SRAM, 无内置 flash

  - LQFP48 / LQFP64 封装, 3.3V 工作电压

+ 内部外设:

  - 定时器 (PWM)

  - DMA

+ 外部接口:

  - GPIO (IO 交换矩阵)

  - SPI (SPI flash)

  - UART

  - I2C

  - ADC

  - USB

  - FMP-P 协议接口

  - 塑料光纤通信接口
