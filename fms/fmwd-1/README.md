# FMWD-1
胖蚊子手环


## 主要设计

+ 与 小米手环 6 部分兼容:

  - 腕带, 磁吸充电器

  - 外形大小

+ 传感器:

  - 运动 (陀螺仪, 加速度计)

  - 温湿度

  - 光照

  - 紫外线

  - 气压

+ 网络: WIFI / BLE

+ 交互:

  - OLED 触摸屏

  - 麦克风

  - 振动电机

+ 大容量锂电池

+ 防水

+ 软件: FMLS 胖蚊子轻蜘蛛

+ 作为普通手环功能的补充:

  - 可运行手环小程序 (WebAssembly)

    类似浏览器的 API 环境

  - FMLH 小蚊 (AI 小助手) she-os
