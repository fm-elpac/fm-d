# PM3P-10s
胖喵人体 3D 扫描台


## 主要设计

+ 圆柱形, 高 2.0m, 直径 1.5m

+ 单摄像头 (手机) 螺旋扫描

+ 预计成本: 约 700 CNY (仅供参考)

+ 3D 建模使用算法: 光流


## 主要组件

+ 六边形木框架

+ 黑色/白色摄影布

+ 圆形地毯 (白色) 直径 140cm

+ 光轴丝杆 (2m) 及直线轴承滑块

+ 顶部/底部 LED 照明模块 (视频定时闪烁)

+ 万向轮

+ 3D 打印件:

  - 导轨, 齿轮轨, 顶部/底部滑块

  - 手机夹座

+ 步进电机及驱动器, 主控板, 电源 (12V)
