# PMCD-1
胖喵防晒液冷服


## 主要设计

+ 使用场景: 夏天 (高温)

+ 续航时间: 8 小时

+ 散热原理: 蒸发散热 (水)

+ 最大散热功率: 500W


## 主要组件

+ 电动车头盔 (白色)

+ 液冷帽

+ 液冷上衣

+ 液冷裤子

+ 液冷手套

+ 液冷鞋

+ 液冷背包: 主要支持组件

+ 蝴蝶翅膀: 扑翼蒸发散热器

+ 小裙子: 被动液冷散热片


## 衣服分层结构

1. 铝箔反光层

2. 外层布料

3. 棉花保温层

4. 水管循环层

5. 内层布料


## 液冷背包

+ 18650 锂电池组

+ 控制电路板

+ 温度传感器

+ 流速传感器

+ 离心式水泵

+ 循环水瓶

+ 蒸发水瓶


## 蝴蝶翅膀

扑翼蒸发散热器.

+ 薄铝板基板

+ 竹签支架

+ 无刷电机动力机构

+ 散热循环管

+ 海绵蒸发层

+ 滴水管路

+ (可选) 薄膜太阳能电池板
