## 1 计算机的分类

计算机: 计算的机器, 信息处理的工具.

按工作原理分类:

+ 机械计算机 / 水流计算机 / 空气计算机

+ 继电器计算机 / 人肉计算机 (《三体》)

+ 模拟电子计算机 (运算放大器)

  加, 减, 乘, 除, 积分, 微分

+ **数字电子计算机** (重点)

  电子管 / 晶体管, 二进制 / 三进制

+ 量子计算机

+ 光子计算机

+ 分子计算机

按用途分类:

+ 个人计算机 (PC): 台式机, 笔记本, 手机, 平板

+ 服务器 (集群, 机房, 机架)

+ 路由器

+ 超级计算机 (E 级)

+ 工业控制计算机

+ 嵌入式 (单片机)
